# Vim

## Vim 的三种模式

- Normal 模式
- Insert 模式
- Command-Line 模式

## Vim 的三种模式切换

- Normal 模式 -> Insert 模式：`i`，`a`，`o`，`I`，`A`，`O`
- Normal 模式 -> Command-Line 模式：`:`，`/`，`?`
- Insert 模式 -> Normal 模式：`Esc`
- Command-Line 模式 -> Normal 模式：`Esc`
